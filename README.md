# Python Fundamentals

파이썬 기초를 넘어 동작 원리나 깊이 있는 개념들을 직접 코드로 짜보면서 정리.
단순 암기보다는 '왜 이렇게 돌아가는지' 이해하는 것에 집중했다. 여기서 다진 기본기로 나중에 어떤 프로젝트든 자연스럽게 적응하는 것이 목표.

## 학습 기록 (섹션별)

* **`02_class/`** : 클래스 심화 (네임스페이스, 클래스/정적 메소드 구분 등)
* **`03_magic_method/`** : 매직 메서드 (내장 기능 오버라이딩, 커스텀 객체 구현)
* **`04_sequence/`** : 시퀀스 (메모리 구조 차이, 가변/불변형 다루기)
* **`05_first_class/`** : 일급 함수 (클로저, 데코레이터로 공통 로직 분리하기)
* **`06_concurrency/`** : 병행성/비동기 (제너레이터, 코루틴, AsyncIO 실습)

## 환경
* Python 3.13.2 (강의는 3.7 기준으로 설명)

## 커밋 규칙
